Паук представляет из себя очередь задач, по мере выполнения которых, производится обработка полученных из сети данных. Он написан таким образом, что параллелится только работа с сетью, а обработка данных выполняется в один поток. В качестве жертвы паука был выбран интернет-магазин http://www.regard.ru. Чтобы случайно не убить сайт большим количеством запросов, парсится только информация о товарах из раздела "Мобильные телефоны". Для хранения данных используется база данных MongoDB.
