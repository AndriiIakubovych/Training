Разработать приложение "убегающий статик". Суть приложения: на форме расположен статический элемент управления. Пользователь пытается подвести курсор мыши к "статику", и, если курсор находится близко со "статиком", элемент управления "убегает". Предусмотреть перемещение "статика" только в пределах формы.
