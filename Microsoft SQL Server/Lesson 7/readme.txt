Создать базу данных "Брачное Агентство" (варианты: "Кадровое агенство").
1) Хранить информацию о тех людях, которые хотят найти свою половину (ФИО, возраст, пол, страна, город, зарплата, цвет кожи, внешние характеристики (рост, вес, цвет волос и т. д), требования к партнёру (какой он должен быть), профессия, хобби, путь к фотографии на жёстком диске (если такая имеется).
2) Реализовать триггеры.
3) Нужно чтобы была следующая функциональность, реализованная через сохраннёные процедуры: показ партнёров из страны, заданной в качестве параметра, показ партнёров по зарплате,  показ партнёров по внешним характеристикам, показ партнёров по комбинации параметров.
4) Добавление партнёров, удаление партнёров, изменение партнёров.
5) Те люди, которые находят пару, должны оставаться в базе. Они переносятся в другую таблицу  (либо остаются в той же с выставленным критерием "нашёл").
