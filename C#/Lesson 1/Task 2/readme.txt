Начальный вклад в банке равен 1000 грн. Через каждый месяц размер вклада увеличивается на P процентов от имеющейся суммы (P — вещественное число, 0 < P < 25). Значение P программа должна получать у пользователя. По данному P определить, через сколько месяцев размер вклада превысит 1100 грн., и вывести найденное количество месяцев K (целое число) и итоговый размер вклада S (вещественное число).
