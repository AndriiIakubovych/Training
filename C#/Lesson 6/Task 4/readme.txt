Создать обобщённый класс точки в 2-х мерном пространстве Point2D<T>, координаты точки могут быть любого типа. В классе предусмотреть поле для хранения значения координаты z. Реализовать в классе:
a) конструктор с параметрами, который принимает начальные значения для координат точки;
b) метод ToString().
