Разработать собственный структурный тип данных для хранения целочисленных коэффициентов A и B линейного уравнения A*X + B*Y = 0. В классе реализовать статический метод Parse, которые принимает строку со значениями коэффициентов, разделённых запятой или пробелом. В случае передачи в метод строки недопустимого формата генерируется исключение FormatException.
