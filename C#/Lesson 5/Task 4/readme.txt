Интерфейс INotifyPropertyChanged пространства имён System.ComponentModel определяет, что наследник содержит событие PropertyChanged, оповещающее об изменении свойств объекта. Изучить данный интерфейс, используя MSDN Library. Проанализировав работу общественных библиотек разработать классы: Author, Book, Client, LibraryCard, Catalogue. Описать их поля, поля инкапсулировать свойствами и для классов реализовать интерфейс INotifyPropertyChanged. Создать диаграмму классов.
