Разработать класс Fraction, представляющий простую дробь. В классе предусмотреть два поля: числитель и знаменатель дроби. Выполнить перегрузку следующих операторов: +, -, *, /, ==, !=, <, >, true и false.
Арифметические действия и сравнение выполняются в соответствии с правилами работы с дробями. Оператор true возвращает true если дробь правильная (числитель меньше знаменателя), оператор false возвращает true если дробь неправильная (числитель больше знаменателя).
Выполнить перегрузку операторов, необходимых для успешной компиляции следующего фрагмента кода:
Fraction f = new Fraction(3, 4);
int a = 10;
double d = 1.5;
Fraction f1 = f * a;
Fraction f2 = a * f;
Fraction f3 = f + d;
