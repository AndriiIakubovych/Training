Дан массив чисел. Написать функцию, которая получает массив, элементы которого являются суммами предыдущих элементов исходного массива (например, если исходный массив равен: { 1, 3, 6, 4 }, то новый массив будет: { 1, 1 + 2, 1 + 2 + 6, 1 + 2 + 6 + 4 }). Функция не должна менять исходный массив, а должна получать новый на выходе.
