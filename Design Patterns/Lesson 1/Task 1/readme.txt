Спроектировать универсальный каркас многодокументного редактора. Редактор должен представлять основные функции работы с документом: создание, открытие, сохранение, сохранение под новым именем, печать, закрытие. Предложенный объектно-ориентированный дизайн каркаса редактора должен без изменений использоваться для разработки редакторов документов различных типов.
