Написать скрипт, который находит переменные, содержащие числа по убыванию и производит с ними операции. Для трёх переменных: наибольшую умножает на 100, следующую на 10, последнюю не трогает. Для четырёх переменных: наибольшую умножает на 1000, следующую на 100, следующую на 10, последнюю не трогает. Для пяти переменных: наибольшую умножает на 10000, следующую на 1000, следующую на 100, следующую на 10, последнюю не трогает.
