Написать программу "Проводная система", которая будет состоять из следующих элементов: переключатели (1 (ON), 0 (OFF)), входы (A (AND), O(OR), N(NOT), X(XOR)), лампочка (@). Выходы являются логическими операторами, они преобразуют сигналы с помощью дискретной логики. Программа должна выводить 1 или 0 в зависимости от того, какой сигнал будет на ламповом элементе. Необходимо реализовать проводную систему в виде структуры данных, содержащей минимум 10 уровней. Переключающие элементы должны присутствовать как минимум на 2-х разных уровнях.
