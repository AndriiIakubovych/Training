Установить NodeJS. Определить форму с двумя полями для ввода имени и возраста пользователя. С помощью обработчика нажатия перехватить отправку этой формы. Получить значения её полей и сериализовать в объект json, который затем отправить на сервер с помощью AJAX на адрес "/data". Получить ответ сервера.
