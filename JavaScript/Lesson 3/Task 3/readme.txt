В некоторых языках программирования существует объект "строковый буфер", который аккумулирует внутри себя значения. Его функционал состоит из двух возможностей: добавление значения в буфер, получение текущего содержимого. Реализовать строковый буфер на функциях в JavaScript, со следующим синтаксисом:
а) создание объекта: var buffer = makeBuffer();
б) вызов makeBuffer должен возвращать такую функцию buffer, которая при вызове buffer(value) добавляет значение в некоторое внутреннее хранилище, а при вызове без аргументов buffer() – возвращает его.
Добавить буферу метод buffer.clear(), который будет очищать текущее содержимое буфера.
