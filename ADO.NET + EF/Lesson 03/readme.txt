Разработать приложение ADO.NET, реализующее работу с базой данных учебного заведения. Приложение должно обеспечивать добавление, удаление, просмотр и модификацию информации о: студентах заведения, преподавателях заведения, учебных аудиториях, читаемых предметах, составах групп, расписании групп, расписании преподавателей, учебных планах, расписании звонков.
Работу приложения основать на фабричном конструктивном шаблоне. Использовать асинхронные механизмы доступа к данным для оптимизации производительности работы приложения. Использовать конфигурационные файлы для соединения с базой данных.
