Создать приложение для хранения и редактирования пользователей. Необходимо хранить следующие данные о пользователе: логин, пароль, адрес, телефон, признак админа (логическая переменная). При добавлении нового пользователя, необходимо проверять, что логин с таким именем свободен. Для защиты пароля использовать функцию GetHashCode у строки.  Список пользователей отображается в ListBox. При выполнении двойного щелчка над записью, содержащей пользователя, открывается вторая форма для редактирования. Реализовать добавление и удаление пользователей. Добавление пользователя реализуется в новом окне. Также реализовать вывод всех зарегистрированных пользователей с фильтрацией администраторов.
