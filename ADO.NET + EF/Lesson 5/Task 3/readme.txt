Разработайте класс телефонный справочник. Для хранения экземпляров класса воспользуйтесь любой коллекцией. Напишите программу, которая вычитывает данные из файла, сохраняет их, а также позволяет просматривать записи, сортировать их по имени или номеру телефона, редактировать записи, добавлять и удалять их. Для сохранения данных в файл воспользоваться сериализацией.
