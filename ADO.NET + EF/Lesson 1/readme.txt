Создать базу данных Продажи. Создать внутри базы данных три таблицы:
a) таблица "Покупатели" (столбцы: идентификатор, имя, фамилия);
b) таблица "Продавцы" (столбцы: идентификатор, имя, фамилия);
c) таблица "Продажи" (столбцы: идентификатор сделки, идентификатор покупателя, идентификатор продавца, сумма сделки, дата сделки);
Наполнить таблицы данными, задать правила ссылочной целостности. Реализовать WinForms приложение, позволяющее пользователю выбрать название таблицы из базы данных (например, с помощью выпадающего списка), в результате выбора таблицы приложение должно отображать содержимое данной таблицы на форму.
