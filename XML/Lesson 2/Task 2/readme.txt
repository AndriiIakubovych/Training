Создать параметрическую сущность, которая описывает использование элемента FullAddress, который может иметь следующие атрибуты:
Country – обязательный атрибут для описания информации о стране, который имеет фиксированный набор значений (например, Украина, Россия, Белоруссия и США);
Town – обязательный артибут для сохранения информации о городе;
Street – необязательный атрибут для сохранения данных об улице, доме и квартире (если есть).
