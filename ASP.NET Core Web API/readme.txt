Создать приложение на подобии google-календаря. Клиент на последней версии Angular, сервер - на ASP.NET Core Web API. Функциональность: имеется авторизация и регистрация; все события локальны для каждого пользователя; отображение событий только в одном режиме - в виде месяца; события состоят только из текста; циклических событий нет; события можно добавлять, редактировать и удалять.
