Реализовать консольное игровое приложение "успел, не успел", где будет проверяться скорость реакции пользователя. Программа должна подать сигнал пользователю в виде текста, и пользователь должен будет нажать кнопку на клавиатуре, после нажатия пользователь должен увидеть, сколько миллисекунд ему потребовалось, чтобы нажать кнопку.
