Создать класс Bank в котором будут следующие свойства: int money, string name, int percent. Построить класс так, чтобы при изменении одного из свойств, класса создавался новый поток, который записывал данные о свойствах класса в текстовый файл на жёстком диске. Класс должен инкапсулировать в себе всю логику многопоточности.
