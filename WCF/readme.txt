Написать Web-службу, которая будет про протоколу SOAP предоставлять следующую функциональность органайзера: добавить заметку на опрелелённую дату, удалить\изменить заметку (выставить её статус), и выдать список задач на определённую дату. Данные хранятся на сервере в базе данных, к базе имеет доступ только web-служба. К этой службе должны быть два клиента: "тонкий" на web и "толстый" – десктопное приложение. Сервис работает по протоколу SOAP и предоставляет свой контракт в WSDL.
