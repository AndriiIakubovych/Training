Написать программу для просмотра графических файлов. Требования: вывод красивого логотипа при запуске программы (можно с анимацией), формирование списка для просмотра путём добавлением нужных папок (поиск в папках может проходить только на верхнем уровне или на всю глубину дерева), отображение всех картинок в виде уменьшенных значков, возможность полноэкранного просмотра (последовательно или случайно), слайдшоу с настройками эффектов.
